# PharmaSafe Frontend

## About
This is the frontend of the PharmaSafe ERP (Pharmacy Inventory and Management System).

## Tech Stack
- React
- TypeScript
- Vite
- Vanilla CSS

## Features
- Dashboard
- Medicines Management
- Suppliers Management
- Customers Management
- Doctors Management
- Purchase Orders
- Medicine Batches
- Prescriptions
- Sales
- POS / Checkout

## Run the Project

```bash
npm install
npm run dev
```
